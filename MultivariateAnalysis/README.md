# Polycystic Ovary Syndrome (PCOS) Analysis and Predictive Modeling

## Overview

This project applies multivariate statistical techniques and machine learning methods to the analysis of **Polycystic Ovary Syndrome (PCOS)** clinical data. The objective is to identify relevant patterns associated with the condition and develop predictive models capable of distinguishing between patients with and without PCOS.

The dataset contains clinical, hormonal, anthropometric, and fertility-related variables collected from patients across multiple hospitals in Kerala, India.

---

## Dataset

- **Observations:** 541 patients
- **Variables:** 40 features
- **Data Types:**
  - 29 numerical variables
  - 11 categorical variables
- **Target Variable:** PCOS (Yes/No)
